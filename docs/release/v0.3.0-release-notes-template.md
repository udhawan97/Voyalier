## Voyalier v0.3.0

Voyalier is a local-first travel workspace: your trips, documents, and keys stay
on your device. This is the first build that can update itself — install it once
and future updates arrive in the app.

### Install once

Download the build for your machine, then open it:

- **macOS (Apple Silicon):** open the `.dmg` and drag Voyalier to
  **Applications** (updates need it installed there). The app isn't yet
  Apple-notarized, so the first launch needs **System Settings → Privacy &
  Security → Open Anyway** (macOS 15+ no longer offers right-click → Open).
- **Windows:** run the `-setup.exe`. SmartScreen may warn on an unrecognized
  publisher — choose **More info → Run anyway**.

Verify your download (optional): match its SHA-256 against `SHA256SUMS-<os>.txt`,
and the build carries verifiable provenance (`gh attestation verify`).

### Updating from here

Voyalier checks GitHub for updates only after you allow it (a one-time ask on
first launch) or when you click **Check for updates** in the Updates panel. Only
release metadata is fetched — nothing about you or your trips is sent. Updates
are signature-verified before they install.

- **macOS:** the update downloads, then you click **Restart Voyalier** to finish.
- **Windows:** the app closes, updates, and reopens (under a minute).

### Your data is safe across updates

Trips, documents, and your encryption key live in the OS data directory
**outside** the app bundle, so updating — or reinstalling if a launch ever fails
— keeps everything. Voyalier also takes a database backup before each update.
