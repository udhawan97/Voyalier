---
title: Getting started
description: Run the Voyalier foundation locally.
---

Requirements are Node.js 24+, pnpm 11+, and the stable Rust toolchain.

```bash
git clone https://github.com/udhawan97/Voyalier.git
cd Voyalier
make bootstrap
make dev
```

The React interface runs at `127.0.0.1:5173`, with the local Axum API at `127.0.0.1:8787`.

Open a trip to try the Phase 2 features: fetch official advice and a weather outlook, search the trip, browse and download offline city packs, and use AI assist. For on-device assist, install [Ollama](https://ollama.com) and pull a model (for example `ollama pull llama3.2`); for cloud assist, add an OpenAI or Anthropic key under **AI providers** (it is stored in your OS keychain). Every AI request shows a full preview before anything is sent.

Run `make check` before submitting changes.
