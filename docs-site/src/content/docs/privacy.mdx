---
title: Privacy and trust
description: How Voyalier protects sensitive trip data.
---

Voyalier is local-first, but live travel research is not automatically offline.

Before cloud AI use, the product must show which excerpts will leave the device and apply sensitive-field redaction. Provider keys belong in the operating-system credential vault, never browser storage.

Documents, websites, model output, and provider output remain untrusted until validated. Extracted facts never silently become accepted trip facts.
