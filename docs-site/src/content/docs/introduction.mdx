Voyalier is a local-first travel intelligence workspace. It turns trip constraints, source-backed research, and messy confirmations into a living Smart Blueprint.

![How the Voyalier trip workspace works: a trip moves from setup through gathering sourced evidence, reviewing extracted facts, deterministic planning, offline preparation, travel-day use of the Today view, optional consented AI help, and redacted sharing.](/Voyalier/assets/voyalier-trip-workspace-flow.svg)

Phases 1–2 are functional as a running app: deterministic planning, consent-gated official advice and weather, trip search, offline city packs, and BYOK AI assist (on-device Ollama plus OpenAI/Anthropic). Phase 3 additions already include persona recommendations, an offline Today view, a consent-gated map, and an encrypted vault. There is no signed installer yet — run it from source; packaging and notarization remain Phase 3 release work.

## Product promise

- Useful deterministic planning without an AI key.
- Optional on-device or BYOK cloud intelligence.
- Sources, freshness, confidence, and unknown states attached to factual claims.
- Explicit review before extracted document facts change a trip.
- Redaction before sharing.

## Non-goals

Voyalier does not initially book travel, guarantee the cheapest inventory, scrape restricted sites, or issue authoritative visa or safety determinations.
