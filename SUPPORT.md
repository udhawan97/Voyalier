# Support

Voyalier is a **source-only public beta** (v0.4.0). It is built in the open and
runs from source; signed, notarized installers are still blocked on paid
certificates.

- Use GitHub Issues for reproducible bugs and feature proposals.
- Use GitHub Discussions for design and product questions once Discussions are enabled.
- Report vulnerabilities privately through GitHub Security Advisories.

Never post real travel documents, identity details, API keys, confirmation numbers, or private itineraries in a public issue.
